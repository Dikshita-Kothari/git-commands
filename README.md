# Commands (All the important once)
  - Check version
```
git --version
```
  - Name and Email
  ```
git config --global user.name ".."
```
```
git config --global user.email ".."
```
  - Initialization
```
git init
```
  - Staging area
```
git status
```
```
git add .
```
  - Commit
```
git commit -m "..."
```
  - Push
```
function test() {
  console.log("notice the blank line before this function?");
}
```
  - Pull
